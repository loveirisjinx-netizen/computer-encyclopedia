Computer History

Old Computer

Early computing machines that filled entire rooms.

Floppy Disk

A legacy magnetic storage medium used in the 80s and 90s.

Old Laptop

The first portable computers, much heavier than today's versions.

Vintage Computer

Classic personal computers from the early era of home computing.

Core Computer Parts

CPU

The central processing unit—the "brain" of the computer.

RAM

Short-term memory for active programs and data.

Motherboard

The main circuit board that connects everything together.

GPU

Handles graphics, video rendering, and gaming performance.

Storage Devices

SSD

Fast, modern storage with no moving parts.

Hard Drive (HDD)

High-capacity storage using spinning magnetic platters.

NVMe

Ultra-fast storage that plugs directly into the motherboard.

Networking

Router

Connects your local network to the internet.

Server

A powerful computer that provides data to other computers.

Ethernet Cable

Used for high-speed wired internet connections.

Input Devices

Keyboard

Used to input text and commands into the computer.

Mouse

A pointing device used to interact with the screen.
